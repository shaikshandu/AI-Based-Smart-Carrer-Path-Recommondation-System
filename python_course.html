Module 4: Control Flow & Conditional Statements

  • If Statement:

    • In Python and other programming languages, the if keyword is used to check whether a condition is true. If the condition is true, the block of code inside the if statement will execute. Remember that indentation is very important in Python.

    Syntax:

    
# syntax
if condition:
    # code runs if condition is True
  • If Else Statement:

    • If the condition is true, the first block will execute. If the condition is false, the code inside the else block will execute.

    Syntax:

    
# syntax
if condition:
    # runs if True
else:
    # runs if False
  • If Elif Else Statement:

    • When we have multiple conditions to check, we use elif. Python checks conditions one by one. Once a condition becomes true, it stops checking further.

    Syntax:

    
# syntax
if condition1:
    # runs if condition1 is True
elif condition2:
    # runs if condition2 is True
else:
    # runs if all conditions are False
  • For Loop:

    • A for loop is used to iterate over a sequence such as a list, tuple, string, or range. It repeats a block of code multiple times.

    Syntax:

    
# syntax
for variable in sequence:
    # code runs for each item
  • While Loop:

    • A while loop executes a block of code as long as the condition remains true. Be careful to avoid infinite loops.

    Syntax:

    
# syntax
while condition:
    # code runs while condition is True
  • Loop Control Statements:

    • Python provides special keywords to control loops: break – Stops the loop completely. continue – Skips the current iteration. pass – Does nothing (placeholder).

    Syntax:

    
for i in range(5):
    if i == 3:
        break
    print(i)

    Module 5: Data Structures

  • Lists:

    • A List is a collection data type in Python which is ordered and changeable (mutable). Lists allow duplicate values and are written inside square brackets []. We use lists when we want to store multiple items in a single variable.

    List Syntax

    
# Creating a list
my_list = [1, 2, 3, 4]

# Accessing elements
print(my_list[0])

# Adding element
my_list.append(5)
  • Tuples:

    • A Tuple is similar to a list but it is immutable (cannot be changed after creation). Tuples are written using parentheses (). They are used when the data should remain constant.

    Tuple Syntax

    
# Creating a tuple
my_tuple = (10, 20, 30)

# Accessing elements
print(my_tuple[1])
  • Sets:

    • A Set is an unordered collection that does not allow duplicate values. Sets are written inside curly brackets {}. They are mainly used to remove duplicates and perform mathematical operations like union and intersection.

    Set Syntax

    
# Creating a set
my_set = {1, 2, 2, 3}

# Output will remove duplicates
print(my_set)
  • Dictionaries:

    • A Dictionary stores data in key-value pairs. It is written using curly brackets {} with keys and values separated by a colon. Dictionaries are very useful for structured data and real-world datasets.

    Dictionary Syntax

    
# Creating a dictionary
student = {
    "name": "Farhan",
    "age": 21
}

# Accessing value
print(student["name"])

# Adding new key-value pair
student["grade"] = "A"
  • Nested Data Structures:

    • Nested data structures means storing one data structure inside another. For example, a list of dictionaries. This is very common in data science when handling datasets.

    Nested Structure Syntax

    
data = [
    {"name": "Alice", "score": 85},
    {"name": "Bob", "score": 90}
]

print(data[0]["name"])

    Module 6: Functions & Lambda Expressions

  • Functions:

    • A function is a block of reusable code that performs a specific task. Functions help in organizing code, avoiding repetition, and improving readability. In Python, functions are defined using the def keyword.

    Syntax:

    
# syntax
def function_name(parameters):
    # code block
    return result
  • Calling a Function:

    • After defining a function, we need to call it to execute the code inside it.

    Example:

    
def greet():
    print("Hello, Welcome to Python!")

greet()
  • Function with Parameters:

    • Parameters allow us to pass data into a function. This makes functions more flexible and reusable.

    Example:

    
def greet(name):
    print("Hello", name)

greet("Farhan")
  • Return Statement:

    • The return keyword is used to send a result back from a function. Without return, the function only executes but does not give back a value.

    Example:

    
def add(a, b):
    return a + b

result = add(5, 3)
print(result)
  • Default Parameters:

    • Python allows default values for parameters. If no value is passed while calling the function, the default value will be used.

    Example:

    
def greet(name="Guest"):
    print("Hello", name)

greet()
greet("Farhan")
  • Lambda Functions:

    • A lambda function is a small anonymous function defined using the lambda keyword. It is used for short, simple operations. Lambda functions can have multiple parameters but only one expression.

    Syntax:

    
# syntax
lambda arguments: expression

    Example:

    
add = lambda a, b: a + b
print(add(4, 6))
  • Why Functions Are Important in Data Science:

    • Functions help in: - Reusing data processing steps - Creating reusable analysis pipelines - Writing clean and modular code - Avoiding repetition in large projects

    Module 7: Object-Oriented Programming (OOP)

  • Introduction to OOP:

    • Object-Oriented Programming (OOP) is a programming paradigm that organizes code using objects and classes. It helps in building scalable, reusable, and organized programs. In Python, almost everything is an object.

  • Class and Object:

    • A class is a blueprint for creating objects. An object is an instance of a class. We use the class keyword to define a class.

    Syntax:

    
# syntax
class ClassName:
    def __init__(self, parameters):
        self.attribute = parameters

# creating object
obj = ClassName(value)
  • Constructor (__init__ method):

    • The __init__ method is a special method that runs automatically when an object is created. It is used to initialize object attributes.

    Example:

    
class Student:
    def __init__(self, name, age):
        self.name = name
        self.age = age

student1 = Student("Farhan", 21)
print(student1.name)
  • Instance Methods:

    • Instance methods are functions defined inside a class. They operate on the object’s data using the self keyword.

    Example:

    
class Student:
    def __init__(self, name):
        self.name = name

    def greet(self):
        print("Hello, my name is", self.name)

student1 = Student("Farhan")
student1.greet()
  • Inheritance:

    • Inheritance allows one class to inherit properties and methods from another class. It helps in code reuse and creating hierarchical relationships.

    Example:

    
class Person:
    def greet(self):
        print("Hello")

class Student(Person):
    pass

s = Student()
s.greet()
  • Encapsulation:

    • Encapsulation means restricting direct access to some data. In Python, we use a double underscore (__) to make attributes private.

    Example:

    
class BankAccount:
    def __init__(self, balance):
        self.__balance = balance

    def get_balance(self):
        return self.__balance

account = BankAccount(1000)
print(account.get_balance())
  • Why OOP is Important in Data Science:

    • OOP helps in: - Building machine learning models as objects - Organizing large projects - Writing reusable and maintainable code - Creating data processing pipelines

    Module 8: NumPy for Data Science

  • Introduction to NumPy:

    • NumPy (Numerical Python) is a powerful library used for numerical computations. It provides support for large multi-dimensional arrays and mathematical functions. NumPy is faster and more efficient than Python lists for numerical operations.

  • Creating NumPy Arrays:

    • NumPy arrays are created using the array() function. Arrays can be one-dimensional or multi-dimensional.

    Example:

    
import numpy as np

arr = np.array([1, 2, 3, 4])
print(arr)
  • Array Attributes:

    • NumPy arrays have useful attributes such as shape, size, and data type. These help in understanding the structure of the data.

    Example:

    
print(arr.shape)
print(arr.size)
print(arr.dtype)
  • Array Operations:

    • NumPy allows element-wise operations such as addition, subtraction, multiplication, and division. Operations are faster compared to normal Python lists.

    Example:

    
arr = np.array([1, 2, 3])

print(arr + 2)
print(arr * 3)
  • Indexing and Slicing:

    • We can access specific elements using indexing and slicing. This is very important in data manipulation.

    Example:

    
arr = np.array([10, 20, 30, 40])

print(arr[0])
print(arr[1:3])
  • Mathematical Functions:

    • NumPy provides built-in mathematical functions such as mean, median, standard deviation, and sum. These are widely used in data analysis.

    Example:

    
arr = np.array([1, 2, 3, 4])

print(np.mean(arr))
print(np.sum(arr))
print(np.std(arr))
  • Why NumPy is Important in Data Science:

    • NumPy is the foundation of libraries like Pandas, Scikit-learn, and TensorFlow. It helps in: - Handling large datasets - Performing fast numerical computations - Preparing data for machine learning

    Module 9: Pandas for Data Analysis

  • Introduction to Pandas:

    • Pandas is a powerful Python library used for data manipulation and data analysis. It provides easy-to-use data structures like Series and DataFrame. Pandas is built on top of NumPy and is widely used in Data Science and Machine Learning.

  • Pandas Series:

    • A Series is a one-dimensional labeled array. It can hold different data types like integers, strings, and floats.

    Example:

    
import pandas as pd

data = pd.Series([10, 20, 30, 40])
print(data)
  • Pandas DataFrame:

    • A DataFrame is a two-dimensional table-like data structure. It contains rows and columns similar to an Excel sheet.

    Example:

    
data = {
    "Name": ["Farhan", "Ali", "Sara"],
    "Age": [21, 22, 20]
}

df = pd.DataFrame(data)
print(df)
  • Reading Data from Files:

    • Pandas can read data from CSV, Excel, and other file formats. This is commonly used in real-world data analysis.

    Example:

    
df = pd.read_csv("data.csv")
print(df.head())
  • Selecting and Filtering Data:

    • We can select specific columns or filter rows based on conditions. This helps in analyzing specific parts of the dataset.

    Example:

    
print(df["Name"])          # Select column
print(df[df["Age"] > 20])  # Filter rows
  • Handling Missing Values:

    • Real-world data often contains missing values. Pandas provides functions to detect and handle missing data.

    Example:

    
print(df.isnull().sum())
df = df.dropna()
  • Basic Data Analysis Functions:

    • Pandas provides powerful built-in functions for analysis. These include mean, median, count, and summary statistics.

    Example:

    
print(df.describe())
print(df["Age"].mean())
  • Why Pandas is Important in Data Science:

    • Pandas is used for: - Cleaning data - Transforming datasets - Preparing data for machine learning - Performing exploratory data analysis (EDA) Almost every Data Science project starts with Pandas.

    Module 10: Data Visualization (Matplotlib & Seaborn)

  • Introduction to Data Visualization:

    • Data Visualization is the process of representing data in graphical form. It helps in understanding patterns, trends, and relationships in data. In Data Science, visualization is used during Exploratory Data Analysis (EDA).

  • Introduction to Matplotlib:

    • Matplotlib is a Python library used for creating static, animated, and interactive visualizations. It is the foundation library for plotting graphs in Python.

    Example (Line Plot):

    
import matplotlib.pyplot as plt

x = [1, 2, 3, 4]
y = [10, 20, 25, 30]

plt.plot(x, y)
plt.xlabel("X-axis")
plt.ylabel("Y-axis")
plt.title("Line Plot Example")
plt.show()
  • Bar Chart:

    • Bar charts are used to compare different categories of data.

    Example:

    
categories = ["A", "B", "C"]
values = [5, 7, 3]

plt.bar(categories, values)
plt.title("Bar Chart Example")
plt.show()
  • Histogram:

    • A histogram shows the distribution of numerical data. It helps in understanding frequency distribution.

    Example:

    
data = [10, 20, 20, 30, 40, 40, 40]

plt.hist(data)
plt.title("Histogram Example")
plt.show()
  • Introduction to Seaborn:

    • Seaborn is built on top of Matplotlib. It provides more attractive and informative statistical graphics. It is commonly used in Data Science for advanced visualizations.

    Example:

    
import seaborn as sns

data = pd.DataFrame({
    "Age": [21, 22, 23, 24, 25],
    "Score": [60, 65, 70, 75, 80]
})

sns.scatterplot(x="Age", y="Score", data=data)
plt.show()
  • Why Data Visualization is Important:

    • Visualization helps in: - Identifying trends and patterns - Detecting outliers - Understanding relationships between variables - Presenting results clearly to stakeholders It plays a key role in Exploratory Data Analysis (EDA).

    Module 11: Statistics for Data Science

  • Introduction to Statistics:

    • Statistics is the science of collecting, analyzing, interpreting, and presenting data. In Data Science, statistics helps in understanding data and making informed decisions. It provides the foundation for Machine Learning algorithms.

  • Types of Statistics:

    • There are two main types of statistics:

    1. Descriptive Statistics: Summarizes and describes the features of a dataset.
    2. Inferential Statistics: Makes predictions or inferences about a population based on a sample.

  • Measures of Central Tendency:

    • These measures describe the center of a dataset.

    Mean: The average value.
    Median: The middle value after sorting.
    Mode: The most frequent value.

    Example:

    
import numpy as np

data = [10, 20, 30, 40]

print(np.mean(data))
print(np.median(data))
  • Measures of Dispersion:

    • These measures describe how spread out the data is.

    Range: Difference between max and min values.
    Variance: Average of squared differences from the mean.
    Standard Deviation: Square root of variance.

    Example:

    
print(np.var(data))
print(np.std(data))
  • Probability Basics:

    • Probability measures the likelihood of an event occurring. It ranges from 0 (impossible) to 1 (certain). Probability is widely used in Machine Learning models.

  • Normal Distribution:

    • A normal distribution is a bell-shaped curve. Many real-world datasets follow a normal distribution. It is important for hypothesis testing and predictive modeling.

  • Correlation:

    • Correlation measures the relationship between two variables. It ranges from -1 to +1. Positive correlation means variables move in the same direction. Negative correlation means variables move in opposite directions.

    Example:

    
import pandas as pd

df = pd.DataFrame({
    "Age": [20, 21, 22, 23],
    "Score": [60, 65, 70, 75]
})

print(df.corr())
  • Why Statistics is Important in Data Science:

    • Statistics helps in: - Understanding data patterns - Making predictions - Building Machine Learning models - Evaluating model performance Without statistics, Data Science is incomplete.

    Module 12: Machine Learning Basics

  • Introduction to Machine Learning:

    • Machine Learning (ML) is a branch of Artificial Intelligence that enables systems to learn from data and make predictions without being explicitly programmed. ML algorithms improve automatically through experience.

  • Types of Machine Learning:

    • There are three main types of Machine Learning:

    1. Supervised Learning: The model is trained using labeled data.
    Examples: Linear Regression, Logistic Regression, Decision Trees.

    2. Unsupervised Learning: The model works with unlabeled data.
    Examples: Clustering (K-Means), Dimensionality Reduction.

    3. Reinforcement Learning: The model learns by interacting with an environment.

  • Supervised Learning Example (Linear Regression):

    • Linear Regression is used to predict continuous values. For example, predicting house prices based on size.

    Example:

    
from sklearn.linear_model import LinearRegression
import numpy as np

X = np.array([[1], [2], [3], [4]])
y = np.array([10, 20, 30, 40])

model = LinearRegression()
model.fit(X, y)

prediction = model.predict([[5]])
print(prediction)
  • Model Training Process:

    • 1. Collect Data
    2. Clean & Prepare Data
    3. Split into Training and Testing Sets
    4. Train the Model
    5. Evaluate Performance
    6. Make Predictions

  • Model Evaluation:

    • To measure model performance, we use evaluation metrics.

    For Regression:
    - Mean Absolute Error (MAE)
    - Mean Squared Error (MSE)

    For Classification:
    - Accuracy
    - Precision
    - Recall
    - F1 Score

  • Overfitting and Underfitting:

    • Overfitting: Model performs well on training data but poorly on new data.
    Underfitting: Model fails to capture patterns in the data.
    A good model balances both.

  • Why Machine Learning is Important:

    • Machine Learning is used in: - Recommendation Systems (Netflix, Amazon) - Fraud Detection - Image Recognition - Chatbots - Predictive Analytics It is one of the most in-demand skills in Data Science.
