Math & Statistics for Data Science

Module 1: Mathematical Foundations

  • Functions

    • A function shows how one variable depends on another. 

    
Linear Function:
f(x) = mx + c

    m = slope c = intercept

  • Summation Notation
    • 
Sum of n numbers:

      n
Σ xi
     i=1
  • Mean (Average)
    • 
Mean (μ) = (Σ xi) / n

    Module 2: Descriptive Statistics

  • Variance
    • 
Population Variance:

σ² = (Σ (xi - μ)²) / N
  • Standard Deviation
    • 
σ = √[(Σ (xi - μ)²) / N]
  • Sample Variance
    • 
s² = (Σ (xi - x̄)²) / (n - 1)
  • Probability
    • 
P(A) = Number of favorable outcomes / Total outcomes
  • Conditional Probability
    • 
P(A | B) = P(A ∩ B) / P(B)

    Module 3: Inferential Statistics

  • Z-Score
    • 
Z = (x - μ) / σ
  • Confidence Interval (Mean)
    • 
CI = x̄ ± Z * (σ / √n)
  • Hypothesis Testing Logic
    • 
If p-value < α → Reject Null Hypothesis
If p-value ≥ α → Fail to Reject Null Hypothesis
  • Standard Error
    • 
SE = σ / √n

    Module 4: Linear Algebra for Machine Learning

  • Vector Representation
    • 
v = [x1, x2, x3, ..., xn]
  • Dot Product
    • 
a · b = Σ (ai * bi)
  • Matrix Representation
    • 
A =

[ a11  a12  a13 ]
[ a21  a22  a23 ]
[ a31  a32  a33 ]
  • Matrix Multiplication
    • 
C = A × B

Cij = Σ (Aik * Bkj)
  • Linear Regression Formula
    • 
y = β0 + β1x

    Final Insight

    Statistics helps measure uncertainty. Linear algebra helps build models. Calculus (future learning) helps optimize them.

    With these mathematical foundations, you are prepared to understand machine learning deeply.
